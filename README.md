# React-Todo

### To Initialize this app follow the following instructions

- Ensure you have nodeJs and npm installed
  
```
$ node --version
$ npm --version
```

- Clone the repository

```
$ git clone git add git@github.com:abdoshbr3322/React-Todo.git
```

- Install dependencies

```
$ npm i
```

- Run The app

```
$ npm start
```
